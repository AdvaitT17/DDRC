Accessibility Statement

Our Commitment

DDRC Mumbai is committed to making this website accessible to all users, including persons with disabilities. We strive to conform to WCAG 2.1 Level AA guidelines.

Accessibility Features

Navigation

  • "Skip to Main Content" link for keyboard users
  • Consistent navigation structure across all pages
  • Breadcrumb navigation for easy orientation
  • Keyboard-accessible menus and controls

Keyboard Shortcuts

  • H - Go to Home page
  • S - Skip to main content
  • F - Site Search
  • T - Back to Top

Chrome/Edge: Alt + Key | Firefox: Alt + Shift + Key | Safari: Alt + Key

Visual Accessibility

  • High contrast color scheme
  • Text resizing support up to 200%
  • Clear visual focus indicators

Screen Reader Support

  • Semantic HTML5 structure
  • ARIA landmarks and labels
  • Descriptive alt text for all images
